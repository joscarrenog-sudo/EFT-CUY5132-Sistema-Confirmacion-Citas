# Sistema Automatizado de Confirmación de Citas Médicas

## Evaluación Final Transversal

**Asignatura:** Comunicaciones Unificadas (CUY5132)

## Integrantes

- José Carreño
- Nombre compañero

## Descripción

Este proyecto implementa una plataforma automatizada para la confirmación de citas médicas de la Clínica Regional "Salud Integral", utilizando tecnologías de Comunicaciones Unificadas, infraestructura Cloud e Inteligencia Artificial.

## Tecnologías utilizadas

- AWS EC2
- Ubuntu Server 24.04
- FreePBX 17
- Asterisk 22
- Kamailio
- RTPEngine
- Docker
- n8n
- Slack
- OpenRouter
- Python

## Funcionalidades

- Confirmación automática de citas.
- Script AGI.
- Captura DTMF.
- Registro de resultados.
- SBC Kamailio.
- Seguridad mediante Fail2Ban.
- Automatización con n8n.
- Agente Virtual con IA.
